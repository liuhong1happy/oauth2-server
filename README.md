# oauth2-server

登录注册第三方用户、创建应用、生成app_key和secret、登录授权页面、获取令牌、获取用户信息、刷新令盘

## 开发环境 

- 开发语言 nodejs
- 数据库 mysql
- web框架 similar-server

## 快速开始

1. 安装mysql
2. 安装nodejs
3. 安装nodejs依赖包

    cd src/backend
    npm install

4. mysql中创建数据库

    oauth-db

5. 快速初始化

    npm run init

6. 运行服务

    npm start

7. 开启测试

    npm run test
